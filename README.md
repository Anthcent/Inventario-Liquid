# Sistema de Ventas de Jabones (POS)

## Instalación
1. Asegúrate de que **XAMPP** esté instalado y ejecutándose (Apache y MySQL).
2. La carpeta del proyecto es: `c:\xampp\htdocs\sistemadejabone`.
3. La base de datos se ha creado automáticamente (`sistemadejabone`).
4. Abre tu navegador en: `http://localhost/sistemadejabone`.

## Características
- **Venta Fraccionada**: Ingresa el monto en Bs (ej. 10 Bs de Jabón) y el sistema calcula la cantidad exacta (ml, litros, onzas) a descontar del inventario.
- **Inventario**: Agrega productos, define sus unidades (Litros, Onzas, etc.) e incrementa el stock fácilmente.
- **Diseño Premium**: Interfaz moderna Oscura con Glassmorphism y totalmente responsiva.

## Tecnologías
- PHP 8.x
- MySQL
- Tailwind CSS (CDN)
- Alpine.js (Lógica Frontend)
